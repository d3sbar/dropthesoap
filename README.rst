# DropTheSoap features:

* One-to-one mapping to XSD schema.

* Simple type modeling. You have not to guess how inherit
  your classes. Simply write in XSD notation.

* On-the-fly WSDL generation.

# Fork edits:

* Support for Python 3.5+

* Support for Async
